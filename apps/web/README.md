# @kiosk24/web

The frontend web application for the kiosk24 project, built with [Astro](https://astro.build/).


## Features

- **History View**: Browse screenshots organized by date and device.
- **Comparison Tool**: Compare screenshots from different dates or configurations.
- **Interactive UI**: Built with Preact for a responsive experience.
- **Server-Side Rendering**: Powered by Astro for optimal performance and SEO.
